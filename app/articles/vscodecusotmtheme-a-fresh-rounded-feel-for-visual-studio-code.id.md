Kalau kamu mulai merasa tampilan **Visual Studio Code** kamu terlihat agak... membosankan, maka **vscodeCusotmTheme** karya [mahmoud-asdf](https://github.com/mahmoud-asdf/vscodeCusotmTheme) adalah jawabannya.

Repo ini bukan sekadar theme biasa — ini adalah **customization package** lengkap yang menggunakan **custom CSS dan JavaScript**.  
Waktu aku coba sendiri, VS Code terasa jadi **segar dan modern**, terutama berkat sentuhan **rounded UI** yang membuat semuanya terlihat lebih halus dan rapi — rasanya seperti pakai editor yang baru.

## ✨ Highlights

- **Rounded UI Design**: Tombol, input, dan elemen lainnya kini punya tampilan yang lebih lembut dan rounded, menghasilkan antarmuka yang jauh lebih bersih dan ramah.
- **Deep Customization**: Atur detail-detail kecil pada UI menggunakan file `css.css` dan `js.js` yang sudah disertakan.
- **Instant Setup**: Cukup copy `settings.json` yang sudah disediakan, install extension custom CSS loader, dan langsung nikmati hasilnya.
- **Super Lightweight**: Tidak ada bloat. Murni peningkatan UI saja.

## 🛠 Quick Start

1. Install extension custom CSS/JS untuk VS Code, seperti:
   - Custom UI Style
   - Custom CSS Hot Reload
   - Custom CSS and JS Loader
2. Copy `settings.json` dari repo ke dalam settings kamu.
3. Update path-nya agar sesuai dengan file lokal kamu jika diperlukan.
4. Aktifkan extension-nya — dan seketika editor kamu terasa seperti baru.
